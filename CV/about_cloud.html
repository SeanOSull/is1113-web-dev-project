Software as a service
Software as a service is a way of delivering centrally hosted applications over the Internet as a service.

SaaS applications are sometimes called web-based software, on-demand software, or hosted software.
The main point of saas is that instead of installing and maintaining software, you simply access it via the Internet, freeing yourself from complex software and hardware management.

The main advantages of SaaS are: Lower initial costs, Painless upgrades and Seamless integration

Platform as a Service (PaaS) - Cloud 9

- Platform as a service also known as PaaS is a cloud computing model in which a certaim provider delivers hardware and software tools which are needed for application development to users via the internet - The main benefit of PaaS is that it liberates users from having to install in-house hardware and software to develop or run a new application. - some advantages of Paas include: (1) PaaS development tools can cut the time it takes to code new apps with pre-coded application components built into the platform, such as workflow, directory services, security features, search, and many more. (2)Develop for multiple platforms—including mobile—more easily. (3)PaaS provides all of the capabilities that you need to support the complete web application lifecycle: building, testing, deploying, managing, and updating.

Infrastructure as a Service (IaaS) - Amazon

Infrastructure as a service (IaaS) is a form of cloud computing that provides virtualised computing resources over the internet. IaaS is one of the three main categories of cloud computing services, alongside SaaS and PaaS. -IaaS helps you avoid the expense and complexity of buying and managing your own physical servers and other data center infrastructure. -IaaS is usful as it Eliminates capital expense and reduces ongoing cost, making it an economical option for start-ups and businesses testing new ideas. -some other advantages of IaaS include : (1)it is a rapid innovation in the way the necessary computing infrastructure can be ready in minutes or hours, rather than the days, weeks or months (2) a cloud service provider can provide security for your applications and data.

Communication as a Service (CaaS) - Gmail

Communications as a Service (CaaS) is an outsourced enterprise communications solution that can be leased from a single vendor. Such communications can include voice over IP, instant messaging, collaboration and videoconference applications using fixed and mobile devices. CaaS has evolved along the same lines as Software as a Service (SaaS).

The CaaS vendor is responsible for all hardware and software management and offers guaranteed Quality of Service. This approach eliminates the large capital investment and ongoing overhead for a system whose capacity may often exceed or fall short of current demand.

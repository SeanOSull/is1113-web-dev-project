Software as a Service (SaaS) - Salesforce

Software as a service (or SaaS) is a way of delivering centrally hosted applications over the Internet—as a service. SaaS applications run on a SaaS provider’s servers. Instead of installing and maintaining software, you simply access it via the Internet, freeing yourself from complex software and hardware management. The provider manages access to the application, including security, availability, and performance. SaaS business applications are usually accessed by users using a thin client via a web browser.

What are the Advantages of SaaS?
High adoption, Lower initial costs, Painless upgrades and Seamless integration

SaaS facilitates remote application hosting and delivery, making this the key advantage of SaaS: painless application access. SaaS customers have no hardware or software to buy, install, maintain, or update. Access to applications is easy—you just need an Internet connection.
